States

Focus

An input field can show a user is currently interacting with it

Loading

An icon input field can show that it is currently loading data

Loading inputs automatically modify the input's icon on loading state to show loading indication

Error

An input field can show the data contains errors

Action

An input can be formatted to alert the user to an action they may perform

Action input do not support <button> because browsers do not implement all display modes for button elements. See chromium and mozilla bug reports for progress on browser implementation.
