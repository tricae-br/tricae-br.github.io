Input

Utiliza input e icon.

Default

<div class="ui input">
  <input type="text" placeholder="Procurar..." />
</div>

Error
Apenas adicionar a class="error".

<div class="ui input error">
  <input type="text" placeholder="Procurar..." />
</div>

Fluid
Apenas adicionar a class="fluid".

<div class="ui input fluid">
  <input type="text" placeholder="Procurar..." />
</div>

Action
Apenas utilizar a class="action".

Buscar
<div class="ui input action">
  <input type="text" placeholder="Procurar..." />
<div class="ui button secondary"> Buscar </div>
</div>
