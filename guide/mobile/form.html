Form Elements

Utiliza grid, dropdown, icon, form, text e input.



Iniciando

- criar uma <div class="ui form"> < /div>
- criar a estrutura baseado no grid
- para rotular os campos, utilizar sempre a tag <label> </label>
- sempre utilizar placeholder
- para elementos input utilizar sempre o type (otimizar a usabilidade mobile)

Exemplo:

Code
<div class="ui form">

  <div class="row">

    <div class="column desktop-4">
      <div class="field">
        <label>Login</label>
        <input type="text" placeholder="seulogin" />
      </div>
    </div>

    <div class="column desktop-4">
      <div class="field">
        <label>Senha</label>
        <input type="text" placeholder="****" />
      </div>
    </div>

    <div class="column desktop-4">
      <div class="field">
        <label>&nbsp;</label>
        <input type="button" class="ui button secondary" value="Entrar" />
      </div>
    </div>

  </div>

</div>


Textarea

Code
    <label>Comentário</label>
    <textarea></textarea>


Input

Code
    <label>Nome</label>
    <input type="text" placeholder="ex.: João dos Santos...">


Checkbox

Code
  <div class="inline field">
    <div class="ui checkbox">
      <input type="checkbox" />
      <label> Checkbox </label>
    </div>
  </div>


Radio Box

Code
<div class="grouped inline fields">

  <div class="field">
    <div class="ui radio checkbox">
      <input type="radio" name="gender" checked="checked">
      <label>Feminino</label>
    </div>
  </div>

  <div class="field">
    <div class="ui radio checkbox">
      <input type="radio" name="gender">
      <label>Masculino</label>
    </div>
  </div>

</div>


Select
Ver mais detalhes em dropdown
Para iniciar o dropdown, é necessário utilizar o script abaixo, já inserido no global.js

Code
<label>Gênero</label>
<select class="ui dropdown">
<option value="Fem">Feminino</option>
<option value="Mas">Masculino</option>
</select>

<script type="text/javascript">
$('select.dropdown')
.dropdown()
;
</script>
