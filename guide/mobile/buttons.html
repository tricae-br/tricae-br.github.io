Botões

Utiliza button. Ver mais em Semantic-UI

Cores

Padrão
Apenas utilizar a class="ui button".

<button class="ui button"> Continuar </button>

Call-to-action
Adicionar a class="primary" ao botão padrão.

<button class="ui button primary"> Ccomprar </button>

Secundário
Adicionar a class="secondary" ao botão padrão.

<button class="ui button secondary"> Detalhes </button>

Tamanhos

Padrão
Adicionar a class="ui button".

<button class="ui button"> Detalhes </button>

Small
Adicionar a class="small".

<button class="ui button small"> Inserir cupom </button>

Large
Adicionar a class="large".

<button class="ui button large"> Continuar comprando </button>

Icon

Default
Apenas adicionar a class="icon" à div "ui button" e o ícone a ser utilizado.

<div class="ui button icon">
   <i class="icon edit"> </i>
</div>

Circular

Default
Apenas adicionar a class="circular" ao Icon Button.

<div class="ui button icon circular">
   <i class="icon edit"> </i>
</div>

Labeled Icon

Adicionar a class="labeled" e a Label ao Icon Button.

Icon Left

Editar
<div class="ui labeled icon button"> <i class="icon edit"> </i> Editar </div>

Icon Right
Adicionar a class="right" e a Labeled Icon Button.

Adicionar
<div class="ui right labeled icon button"> <i class="right icon add"> </i> Adicionar </div>

Group

Horizontal Icon buttons
Apenas adicionar a <div class="ui buttons"> </div> aos Icon Buttons.

<div class="ui buttons">
   <div class="ui icon button">
     <i class="icon edit"> </i>
   </div>
   <div class="ui icon button">
     <i class="icon trash"> </i>
   </div>
   <div class="ui icon button">
     <i class=" icon add"> </i>
   </div>
</div>
